# VagrantLamp
Vagrant Lamp stack for project work

Requirements
------------
* VirtualBox <http://www.virtualbox.com>
* Vagrant <http://www.vagrantup.com>

Usage
-----
### Create Project
  	$ mkdir PROJECTNAME
  	$ PROJECT=yourprojectname vagrant up

### Connecting
	$ vagrant ssh

The server should then be available at yourprojectname.com
